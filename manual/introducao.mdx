# Bem-vindo ao NexamedClinic

O **NexamedClinic** é uma plataforma completa de gestão clínica que permite agendar consultas, gerenciar pacientes, realizar videochamadas, aplicar formulários de anamnese, gerenciar protocolos clínicos com IA e muito mais.

Este manual de usuário foi criado para guiar profissionais de saúde, administradores e equipe clínica no uso de todas as funcionalidades do sistema.

## Requisitos do Sistema

Para utilizar o NexamedClinic, você precisa apenas de:

- Navegador moderno (Chrome, Firefox, Edge, Safari — versões atuais)
- Conexão à internet estável
- Para videochamadas: câmera e microfone habilitados
- Conta de usuário ativa fornecida pelo administrador da clínica

<Tip>
  Recomendamos o uso do **Google Chrome** para a melhor experiência, especialmente nas videochamadas.
</Tip>
